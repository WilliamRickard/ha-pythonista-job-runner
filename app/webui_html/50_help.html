Quick start and troubleshooting
Quick start
  1. Copy the sample Python task below and update the add-on URL and token only if you are not using Ingress.
  2. Run it from Pythonista, then return here and open the new job with View.
  3. Success means the job reaches done and a result zip link appears.
Direct access needs an X-Runner-Token header. Ingress sessions usually provide auth automatically.

Troubleshooting
  • Disconnected: check add-on logs and network/access path.
  • Unauthorized: verify X-Runner-Token and token value.
  • No jobs: copy the sample task, run it, then use the header Refresh once if the list does not update automatically.
