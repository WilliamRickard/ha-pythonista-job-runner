# Screenshot placeholders

This folder is reserved for screenshots referenced by the repository root `README.md`.

Suggested screenshots:

1. `01_addon_store.png`: Home Assistant Add-on Store entry for Pythonista Job Runner (show the install button and description).
2. `02_config_token.png`: Add-on configuration page showing the Access token field (do not include a real token).
3. `03_webui_jobs.png`: Ingress Web UI showing the job list and a completed job.

Guidelines:

- Avoid capturing personal information (hostnames, usernames, IP addresses, tokens).
- Prefer screenshots that fit within a typical phone screen width, as many users will read the README on mobile.
