Help
Pythonista Job Runner
Quick actions
  1. Submit a zip from Pythonista using POST /run.
  2. Use search and state filters to find the job quickly.
  3. Open View for details and download result zip when ready.
If calling directly (outside Home Assistant Ingress), send X-Runner-Token on protected endpoints.
Troubleshooting
  • Disconnected: check add-on logs and network/access path.
  • Unauthorized: verify X-Runner-Token and token value.
  • No jobs: confirm the Pythonista script uploads to /run.
API reference
Core endpoints include copy buttons for faster client setup.
