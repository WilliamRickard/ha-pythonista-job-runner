Pythonista Job Runner
Quick start
  1. Set an access token in the add-on Configuration.
  2. From Pythonista, submit a job by sending a zip file to POST /run.
  3. Watch progress here, then download the result zip from the job details view.
If you call the runner directly (not via Home Assistant Ingress), your client must send the X-Runner-Token header.
API and endpoints
These are the endpoints exposed by the runner. Tap Copy to copy a full URL.
Direct access requires X-Runner-Token unless you are using Ingress.
Troubleshooting
  • If the status shows Disconnected, open the add-on Log tab and check for errors.
  • If you see unauthorised, you are likely using direct access without X-Runner-Token, or the token does not match the add-on Configuration.
  • If jobs never appear, confirm your Pythonista script uploads a zip to /run and stays under max_upload_mb.
