Quick start and troubleshooting
Quick start
  1. Submit a zip from Pythonista using POST /run.
  2. Use Search and state filters to find the job quickly.
  3. Open View to inspect logs and download the result zip.
Troubleshooting
  • Disconnected: check add-on logs and network/access path.
  • Unauthorized: verify X-Runner-Token and token value.
  • No jobs: confirm the Pythonista script uploads to /run.
