# Web UI CSS parts

This folder contains the source CSS split into parts for maintainability.

Do not edit `webui.css` directly. It is generated by `webui_build.py` from the parts in this folder.

## Assembly order

The build is deterministic and uses an explicit ordered list:

- 00_tokens.css
- 10_layout.css
- 20_jobs_table.css
- 30_logs.css
- 40_overlays.css
- 50_responsive.css

## Rules

- No root-relative references (for example `url(/...)` or `@import "/..."`). Ingress runs under a path prefix, so root-relative URLs will break.
- When adding a new part, update `WEBUI_CSS_PARTS` in `webui_build.py` and keep filenames prefixed to reflect ordering.
