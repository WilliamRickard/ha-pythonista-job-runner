# Web UI CSS parts

## Version headers

These READMEs are intentionally versionless. The Web UI version is controlled by `WEBUI_VERSION` in `pythonista_job_runner/app/webui_build.py`.
Do not add a line like `<!-- Version: ... -->` here; the build fails if it appears.

This folder contains the source CSS split into parts for maintainability.

Do not edit `pythonista_job_runner/app/webui.css` directly. It is generated by `pythonista_job_runner/app/webui_build.py` from the parts in this folder, and then inlined into the bundled `webui.html`.

## Assembly order

The build is deterministic and uses an explicit ordered list (`WEBUI_CSS_PARTS`) in `pythonista_job_runner/app/webui_build.py`.
If you add, remove, or rename a part, update that list too.

## Rules

- No root-relative references (for example `url(/...)`, `url("/...")`, `@import "/..."`). Ingress runs under a path prefix, so root-relative URLs will break; the build fails and reports file and line number.
- Keep selectors and variables scoped sensibly. Tokens and shared variables should live in the earliest part(s) to avoid circular overrides.

## Rebuild and check

```bash
python pythonista_job_runner/app/webui_build.py
python pythonista_job_runner/app/webui_build.py --check
```
