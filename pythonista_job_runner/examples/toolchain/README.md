# Toolchain examples

These examples are scaffold placeholders in Phase 1.

When implemented, this track will hold examples that require a toolchain-enabled add-on image, such as C++ and Rust builds. They are intentionally separated from the default-image examples so the main add-on can stay lightweight.

Do not assume the folders in this track are runnable on the default image until the later toolchain phases in the workplan are complete and user-validated.
