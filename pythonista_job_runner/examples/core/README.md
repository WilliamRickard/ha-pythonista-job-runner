# Core examples

These examples are intended for the default lightweight add-on image.

Current state:
- `01_hello_world` is implemented, user-validated, and includes checked-in validation evidence plus deterministic reference files
- `02_live_logs_progress` to `05_requirements_optional` are still scaffold placeholders for later phases

Use `01_hello_world` as the first regression check for both the add-on and the standalone Pythonista runner.
